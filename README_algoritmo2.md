# Algoritmo de Descuento en Factura - Versión Mejorada

## Descripción
Versión mejorada del programa que calcula el descuento del 20% en una factura con validación de entrada y redondeo de decimales.

## Características
✅ Valida que el usuario ingrese un número válido  
✅ Rechaza montos negativos  
✅ Soporta decimales (no solo enteros)  
✅ Redondea a 2 decimales (centavos)  

## Cómo usar
1. Ejecutá el programa.
2. Ingresá el monto bruto cuando te lo pida.
3. Si ingresás algo inválido (texto, número negativo), te pedirá de nuevo.
4. El programa muestra el desglose: monto bruto, descuento y total.
